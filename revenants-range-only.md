## RULES
```
-- Locked to a combat stlye (ranged selected)
-- All the combat training can only be done in Revenant Caves
-- Can't level other combat styles till it's no longer viable
-- Stay as much low level as possible althought gaining HP
-- 0 quests that award any combat xp including range
-- Quests with restrictions to combat styles can't be done
-- All the remaining quests can only be done locked to range with minimal combat xp gained
-- The quest materials can be acquired in GE but only with GP gained in Revenant Caves
-- No trades allowed nor sponsorships
-- No minigames allowed unless for teleport or skilling (fishing, mining, etc)
-- All skills can be trained as long as we have GP from Revenants to continue.
-- Besides uniques the quest rewards do not count for the GP allowance.
-- (more will be added as needed)
```

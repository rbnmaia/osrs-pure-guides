## RULES
```
1. Locked to a combat stlye (magic selected)
2. All the combat training can only be done in Revenant Caves
3. Can't level other combat styles till it's no longer viable
3. Stay as much low level as possible althought gaining HP
4. 0 quests that award any combat xp including magic
5. Quests with restrictions to combat styles can't be done
6. All the remaining quests can only be done locked to range with minimal combat xp gained
7. The quest materials can be acquired in GE but only with GP gained in Revenant Caves
8. No trades allowed nor sponsorships
9. No minigames allowed unless for teleport or skilling (fishing, mining, etc)
10. All skills can be trained as long as we have GP from Revenants to continue.
11. Besides uniques the quest rewards do not count for the GP allowance.
```
> This was an experiment based on Wildinator series.
From tutorial island lvl3 straight into revenants.
